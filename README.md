# Java Unit Converter

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)

A simple, yet powerful command-line utility built with Java to perform common unit conversions. This project is an excellent starting point for beginners learning Java, covering fundamental concepts like user input, control flow, and basic method implementation.

## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

You will need to have a Java Development Kit (JDK) installed on your system to compile and run the application. You can download it from the [Oracle website](https://www.oracle.com/java/technologies/downloads/).

### Running the Application

1.  **Clone the repository** or download the source code to your local machine.
2.  **Open your terminal** and navigate to the directory where the files are located.
3.  **Compile the Java source code** using the Java compiler:
    ```bash
    javac Main.java
    ```
4.  **Run the compiled program** from the terminal:
    ```bash
    java Main
    ```

## ‚öôÔ∏è Usage

Once the application is running, you will be presented with a menu of conversion options:

```
--- Unit Converter Menu ---
1. Celsius to Fahrenheit
2. Fahrenheit to Celsius
3. Kilometers to Miles
4. Miles to Kilometers
```

1.  Enter the number corresponding to your desired conversion.
2.  Follow the prompts to enter the value you wish to convert.
3.  The converted result will be displayed on the screen.
4.  To exit the program, select the "Exit" option from the menu.
